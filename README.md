# Zientzia-Astea-2020


## Izenburua: Aljebra linealaren aplikazio bat: irudiak konprimitzea


## Laburpena

 Irudi bat matrize handi bat da, non irudia osatzen duen pixel bakoitzari bere kolorea zehazten duen matrizearen  osagai bat dagokion. Irudia konprimitzearen arrakasta, elkarren ondoan dauden pixelen kolorearen eskala antzekoa izatearen printzipioan oinarritzen da.  Konpresioa ontzat hartuko da,  jatorrizko eta konprimitutako irudiak  desberdintzeko gai ez garenean

 Balio Singularren Deskonposaketak (SVD)  edozein matrize, zati sinpleagotan banatzen du:  zati hauek  garrantziaren araberako ordenan lortzen dira eta beraz,  esangura gutxieneko zatiak kendu daitezke irudiaren kalitatean galera handirik eragin gabe
 
 SVD-k  Datu-Zientzia arloan paper garrantzitsua jokatzen du eta aplikazio aurreratu askotan erabilgarria da:  Irudien Prozesamendua, Aholku-Ingeniaritza (NetFlix, Amazon), Osagai Nagusien Analisia, Aurpegi-Errekonozimendu Sistemak, Gene-Adierazpena, e.a.
